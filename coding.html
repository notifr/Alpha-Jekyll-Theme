Learn to Code

Learn to Code is a 3 hour workshop for kids aged 8-12 to go from the basics to a working game prototype.

About the Workshop

Learn to Code is an extremely fun and interactive way to to learn the in's and out's of coding, and even leave with a working game designed and programmed by each student.

Each student in this workshop will:

  • Design games using Scratch
  • Learn how to combine creative and logical thinking
  • Think in a programmers mindset
  • Go from being a beginner to leaving with a working prototype

We only have limited spaces for this programme, and we purposly keep the classes small so that everyone get's 1-on-1 help.

Learn to Code

Learn to Code is a 3 hour workshop for kids aged 8-12 to go from the basics to a working game prototype.

About the Workshop

Learn to Code was designed to be fun, engaging and leave ki

  • Design games using Scratch
  • Learn how to combine creative and logical thinking
  • Think in a programmers mindset
  • Go from being a beginner to leaving with a working prototype
